Robots evolved are stored under robots in a folder with the team name.
To see the robots in action against the test robots, use testRobots.bat
Use options.txt to change the options the proyect uses, robot name, teams, etc, please maintain the format.

Instructions:
rename files with extension ".ba2" to ".bat"
unzip robocode to the proyect
unzip jikes to the project
copy rt.jar from the java install directory to the proyect

run, testRobots.bat
